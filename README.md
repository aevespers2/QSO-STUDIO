# QSO-STUDIO

Visual design, inspection, orchestration, and debugging workspace for Quantum State Objects and the broader QSO ecosystem.

## GitHub Pages

The project site is published from the `docs/` directory through `.github/workflows/pages.yml`.

## Product boundary

QSO-STUDIO is the operator and developer interface. It visualizes objects, genomes, messages, provenance, payment flows, freeze points, experiments, and release state without bypassing runtime authorization or safety controls.
